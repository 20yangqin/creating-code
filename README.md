创建您自己的定制设计软件。

### 作业要求

- 至少有两个功能按钮，这些按钮的具体功能（如画笔样式、颜色、动作等）请自行设定；
- 利用这个工具绘制并导出一个美观的示例草图
- 通过readme.md说明这个定制设计软件的功能和使用方法。

✏️使用readme.md进行作业介绍与说明，可通过图文等形式丰富readme.md中的内容

可实现功能：
- 调换3种笔刷，笔刷效果与鼠标移动速度有关
- 选择随机的四种颜色进行绘图
- 清屏
- 更换背景颜色
